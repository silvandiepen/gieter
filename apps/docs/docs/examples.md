# Examples

The repository includes sample projects that are generated by Girk itself.

## Deployable Examples

- `apps/example-basic` exercises archives, sections, tags, redirects, copied files, custom style, and custom script.
- `apps/example-multilang` exercises translated routes, language switching, language-specific project titles, and style overrules.

These projects are built and verified automatically as integration fixtures, and their names are intentionally subdomain-friendly for deployments like `example-basic.girk.dev`.
